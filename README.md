### How to run

> Cloning the repository
 ```bash
  git clone https://github.com/amorimmaria/clocker.git
 ```
 > Accessing project
  ```bash
    cd clocker
  ```
  > Download dependencies
  ```bash
    yarn
  ```
  > Running project
  ```bash
    yarn dev
  ```
